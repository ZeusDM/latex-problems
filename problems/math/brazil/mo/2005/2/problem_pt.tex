Determine o menor número real $C$ para o qual a desigualdade
$$C (x_1^{2005} +
     x_2^{2005} +
     x_3^{2005} +
     x_4^{2005} +
     x_5^{2005})  \le 
                      x_1x_2x_3x_4x_5
                      (x_1^{125} +
                       x_2^{125} +
                       x_3^{125} +
                       x_4^{125} +
                       x_5^{125})^{16}$$
é válida para todos os números reais positivos $x_1, x_2, x_3, x_4, x_5$.
