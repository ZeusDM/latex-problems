Ache todas as listas $(x_1, x_2, \ldots, x_{2020})$ de reais não negativos tais que as seguintes três condições são satisfeitas:
\begin{itemize}
	\item $x_1 \le x_2 \le \ldots \le x_{2020}$;
	\item $x_{2020} \le x_1  + 1$;
	\item existe uma permutação $(y_1, y_2, \ldots, y_{2020})$ de $(x_1, x_2, \ldots, x_{2020})$ tal que $$\sum_{i = 1}^{2020} ((x_i + 1)(y_i + 1))^2 = 8 \sum_{i = 1}^{2020} x_i^3.$$
